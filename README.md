# Shopping Mall Store Management System
//This project is a store management system designed to manage shops within a shopping mall environment. It focuses on organizing and structuring data related to stores, employees, and services.

## Description
The system is being developed as part of my learning journey in programming, with a focus on object-oriented design and clean code structure.
Currently, the project includes the core class structure responsible for representing the main entities of the system.

## Technologies
- Java
- Object-Oriented Programming (OOP)

## Features
- Representation of stores and their attributes
- Management structure for employees
- System organization using classes and relationships
- Initial architecture for future expansion
